# Doctor Appointment App 🏥

A modern, user-friendly mobile application for booking doctor appointments, built with Flutter. This app connects patients with healthcare providers seamlessly.

## 📱 Features

### For Patients
- **User Authentication** - Secure signup/login using email, phone, or social media
- **Doctor Search** - Find doctors by specialty, location, rating, and availability
- **Appointment Booking** - Schedule appointments with preferred time slots
- **Video Consultations** - Conduct online video calls with doctors
- **Prescription Access** - View and download digital prescriptions
- **Medical Records** - Store and manage health records securely
- **Payment Integration** - Pay for consultations via multiple payment methods
- **Reviews & Ratings** - Rate and review doctors after appointments
- **Appointment Reminders** - Push notifications for upcoming appointments
